# Dotfiles
My dotfiles are managed with [YADM](https://yadm.io/docs/getting_started)

## Installation

1. [Install YADM](https://yadm.io/docs/getting_started#)
2. `yadm clone https://github.com/mathiasbynens/dotfiles.git`

# Add a file to YADM

```
yadm add $file
yadm commit -m "Adding $file"
yadm push
```
